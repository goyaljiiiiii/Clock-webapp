# Chronoscape Clock Suite

Chronoscape is a modern, single-page clock dashboard built with vanilla HTML, CSS, and JavaScript.
It combines real-time clocks and practical planning tools in a clean, responsive interface.

## Features
- Timezone-aware primary clock (`Local`, `UTC`, and major regions)
- 12-hour / 24-hour display toggle
- Live date and contextual greeting
- Auto day/night theme based on active timezone hour
- Animated analog clock face
- World clocks
- Stopwatch
- Countdown timer
- Meeting planner with overlap windows across two timezones
- `localStorage` persistence for key preferences

## Getting Started
1. Clone or download the repository.
2. Open `index.html` in a modern browser.

No build step is required.
